# NestBites Backend

NestBites is a production-ready Home Food Delivery Platform built with Spring Boot and MongoDB.

## Tech Stack

- Java 21
- Spring Boot 3
- Spring Security
- JWT Authentication
- MongoDB
- Cloudinary
- Razorpay
- Swagger OpenAPI
- Docker

## Features

- User Authentication (JWT)
- Chef Registration & Approval
- Menu Management
- Order Management
- Cart
- Favorites
- Reviews
- Search
- Notifications
- Payment Integration (Razorpay)
- Image Upload (Cloudinary)
- Admin Dashboard

## Run

```bash
mvn clean install
mvn spring-boot:run
```

Swagger

```
http://localhost:8080/swagger-ui/index.html
```

API Docs

```
http://localhost:8080/v3/api-docs
```
